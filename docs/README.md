## 快速开始

```shell
npm run serve # 本地启动项目
npm run build # 生产环境打包
npm run lint # 执行eslint检查
npm run test:unit # 执行单元测试
npm run docs:dev # 本地启动该文档项目
npm run docs:build # 打包文档项目
```
